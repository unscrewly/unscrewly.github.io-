FAQ

Questions answered honestly

Do MRs need to download anything from the Play Store?
+
No. MR Force is a Progressive Web App. MRs open a link in their browser and tap "Add to Home Screen". It works like a native app — no Play Store or App Store needed. Works on any Android or iPhone.
How many MRs can we add?
+
Unlimited. There are zero per-user fees. Whether you have 3 MRs or 50, the price is the same ₹18,000/year. Add or remove MR accounts any time from the admin panel.
How exactly does the Google Sheets integration work?
+
When you initialize a financial year for an MR, the system calls an external API that automatically creates a dedicated Google Sheet for that MR for that year. When the MR submits monthly sales data, it goes directly into that sheet via API. One sheet per MR per FY — it can never be duplicated.
Can we import our existing doctor and chemist data?
+
Yes. With the Onboarding plan we handle the full import for you. With Standard we provide an import template. Most companies are fully set up within 48 hours of payment.
Who owns the data? Is it secure?
+
You own your data entirely. It's hosted on Supabase with Row Level Security on every table — each MR only sees their own data, only admin sees everything. We never access your data. You can export everything any time.
What if we need a custom feature or modification?
+
We build on this platform actively. Common customizations — custom reports, regional language support, additional fields — can be quoted. WhatsApp us and we'll give you a straight answer within the day.
